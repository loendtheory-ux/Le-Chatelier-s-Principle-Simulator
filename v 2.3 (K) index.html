Le Châtelier's Principle
StressShiftK?Key Fact
➕ Add Reactant→ RightNoQ < K → shifts right
➖ Remove Reactant← LeftNoQ > K → shifts left
➕ Add Product← LeftNoQ > K → reverse speeds up
➖ Remove Product→ RightNoDrives forward — used in industry!
🔥 Increase TempExo:← / Endo:→YESHeat = product(exo) or reactant(endo)
❄️ Decrease TempExo:→ / Endo:←YESONLY stress that changes K!
🔼 Increase Pressure→ Fewer mol sideNoOnly if gas moles unequal
🔽 Decrease Pressure→ More mol sideNoNo effect if equal moles
✨ Add Catalyst↔ No ShiftNoLowers Ea both ways — faster only!
💨 Add Inert Gas↔ No ShiftNoNo effect at constant volume
⚡ Golden Rule
A disturbed equilibrium shifts to partially counteract the stress — never fully undoes it.
🔑 K Facts
ONLY temperature changes K. Concentration, pressure, catalyst, inert gas change the position but never K.
🧮 Q vs K
Q<K → shift RIGHT  |  Q>K → shift LEFT  |  Q=K → equilibrium
